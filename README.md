# 🌐 Ronda Monitoramento — Ficha de Cadastro

Sistema web de cadastro de colaboradores com preenchimento automático e geração de mensagens para WhatsApp.

## 🔗 Acesso

**[Abrir o sistema →](https://SEU-USUARIO.github.io/ronda-monitoramento/)**

> Substitua `SEU-USUARIO` pelo seu nome de usuário do GitHub.

---

## ✨ Funcionalidades

- **Preenchimento automático** — cole o texto do contrato e os campos se preenchem sozinhos
- **Detecção de Contrato Vigia** — quando detectado, exibe campos de Valor e Justificativa para conversão em All Service
- **Copiar para WhatsApp** — gera mensagem formatada com negrito e emojis, pronta para enviar
- **Modelos de Contrato** — acesso rápido aos modelos *All Service* e *Vigia* para enviar ao colaborador
- **Salvar em PDF** — imprime a ficha preenchida diretamente do navegador
- **Funciona offline** — após o primeiro acesso, não precisa de internet

---

## 🚀 Como publicar no GitHub Pages

1. Crie um repositório no GitHub chamado `ronda-monitoramento`
2. Faça upload de todos os arquivos desta pasta
3. Vá em **Settings → Pages**
4. Em *Source*, selecione `main` branch e pasta `/ (root)`
5. Clique em **Save**
6. Aguarde ~1 minuto e acesse o link gerado
